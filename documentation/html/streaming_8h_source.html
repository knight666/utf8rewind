38 /*
39  UAX15-D4. Stream-Safe Text Process
40 
41  This is the process of producing a Unicode string in Stream-Safe Text Format by processing that string
42  from start to finish, inserting U+034F COMBINING GRAPHEME JOINER (CGJ) within long sequences of
43  non-starters. The exact position of the inserted CGJs are determined according to the following algorithm,
44  which describes the generation of an output string from an input string:
45 
46  * If the input string is empty, return an empty output string.
47  * Set nonStarterCount to zero.
48  * For each code point C in the input string:
49  * Produce the NFKD decomposition S.
50  * If nonStarterCount plus the number of initial non-starters in S is greater than 30, append a CGJ to
51  the output string and set the nonStarterCount to zero.
52  * Append C to the output string.
53  * If there are no starters in S, increment nonStarterCount by the number of code points in S; otherwise,
54  set nonStarterCount to the number of trailing non-starters in S (which may be zero).
55  * Return the output string.
56 */
