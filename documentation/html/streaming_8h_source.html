31 /*
32  UAX15-D4. Stream-Safe Text Process
33 
34  This is the process of producing a Unicode string in Stream-Safe Text Format by processing that string
35  from start to finish, inserting U+034F COMBINING GRAPHEME JOINER (CGJ) within long sequences of
36  non-starters. The exact position of the inserted CGJs are determined according to the following algorithm,
37  which describes the generation of an output string from an input string:
38 
39  * If the input string is empty, return an empty output string.
40  * Set nonStarterCount to zero.
41  * For each code point C in the input string:
42  * Produce the NFKD decomposition S.
43  * If nonStarterCount plus the number of initial non-starters in S is greater than 30, append a CGJ to
44  the output string and set the nonStarterCount to zero.
45  * Append C to the output string.
46  * If there are no starters in S, increment nonStarterCount by the number of code points in S; otherwise,
47  set nonStarterCount to the number of trailing non-starters in S (which may be zero).
48  * Return the output string.
49 */
